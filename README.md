# StarEdge

StarEdge 是面向 GNSS 与相邻定位技术的中文研究知识库。每天北京时间 08:00 汇总 5 条近期快报，并深度研读信号捕获、信号跟踪、接收机工程、低成本/手机优化和纯 GNSS 定位五个方向。

所有技术结论尽量追溯至论文原文、标准、官方技术报告或具备实验依据的工程资料。日报只做摘要、分析和工程推演，不替代原始文献。

## 内容结构

- `digests/YYYY/MM/YYYY-MM-DD.md`：每日 5 条快报与 5 篇深读。
- `data/source-catalog.json`：来源去重及定位专题轮换台账。
- `data/automation-state.json`：自动化启用、成功与漏跑状态。

## 深读方向

1. GNSS 信号捕获。
2. GNSS 信号跟踪。
3. 接收机与实时工程实现。
4. 低成本芯片或手机定位优化。
5. SPP、RTK、PPP/PPP-RTK、双天线定向等纯 GNSS 定位算法。
