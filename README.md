# StarEdge

StarEdge 是面向 GNSS 与相邻定位技术的中文研究知识库。每天整理 5 条近期技术快报，并按照“经典基础—基础进阶—定位深入”的顺序精读 3 篇资料。

精读关注“用什么方法解决什么问题”：先讲透一个小而经典的信号处理或接收机机制，再进入更深入的定位算法。内容包含公式推导、原文图表、常见误区、工程实现和复现实验。所有技术结论尽量追溯至论文原文、标准、官方技术报告或具备实验依据的中英文工程资料。

## 内容结构

- `digests/YYYY/MM/YYYY-MM-DD.md`：每日 5 条快报与 3 篇由浅入深的精读。
- `assets/YYYY-MM-DD/`：精读引用并分析的原文图表。
- `data/source-catalog.json`：来源去重、深读方向与定位专题轮换台账。
- `data/automation-state.json`：日报启用、成功与漏跑状态。

## 精读方向

1. GNSS 信号捕获。
2. GNSS 信号跟踪。
3. 接收机与实时工程实现。
4. 低成本芯片或手机定位优化。
5. SPP、RTK、PPP/PPP-RTK、双天线定向等纯 GNSS 定位算法。
